
## 음운 (Phonology)
| 타입 | 음성 |
|---:|:---|
| 모음 | a, i, u, e, o, uo, ou|
| 자음 | p, t, k(kh), c, m, n, s(sh), r, l |

## 철자 규칙
- `k`는 항상 /kh/로 읽음: `k` → 발음 [kh] (kh = ㅋ하)
- `s`는 /ʃ/로 읽음: `s` → [ʃ] (ʃ = 쉬)
- `c`는 항상 /k/ (c = k = ㅋ)
- `p`는 /p̚/으로 읽음 (p̚ = ㅍ)
- `t`는 /t/로 읽음 (t = ㅌ)
- `i`는 '잇' 으로 읽음

### 이외 규칙
- `re` 는 `레` 가 아닌 `리` 로 발음한다.

## 문화말
- `releit` : 만남 인사 / (releio = 리레잇트)
- `teleit` : 헤어짐 인사 / (teleio = 테레잇트)
- `ruon` : 형식적 만남 인사 / (ruon = 루오느)
- `tuon` : 형식적 헤어짐 인사 / (tuon = 투오느)
- `knat` : 고마움 / (knat = 크하나트)
- `snat` : 미안함 / (snat = 쉬나트)


## 대명사
- `kn` : 그 / (khn = 크하느)
- `rt` : 그것 / (rt = 르트)
- `km` : 그녀 / (khm = 크하므)
- `ti` : 그들 / (ti = 트잇)
- `e`  : 나 / (e = 에)
- `ak` : 너(당신) / (ak = 아크하)
- `te` : 우리 / (te = 테)
