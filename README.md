# 이름 미정

## 음운 (Phonology)
| 타입 | 음성 |
|---:|:---|
| 모음 | a, i, u, e, o, uo, ou|
| 자음 | p, t, k(kh), c,  m, n, s(sh), r, l |

## 철자 규칙
- `k`는 항상 /kh/로 읽음: `k` → 발음 [kh] (kh = ㅋ하)
- `s`는 /ʃ/로 읽음: `s` → [ʃ] (ʃ = 쉬)
- `c`는 항상 /k/ (c = k = ㅋ)
- `p`는 /p̚/으로 읽음 (p̚ = ㅍ)
- `t`는 /t/로 읽음 (t = ㅌ)
- `i`는 '잇' 으로 읽음

## 문화말
- `releio` : 만남 인사 / (releio = 리레이오)
- `teleio` : 헤어짐 인사 / (teleio = 테레이오)
- `ruon` : 형식적 만남 인사 / (ruon = 루오ㄴ)
- `tuon` : 형식적 헤어짐 인사 / (tuon = 투오ㄴ)
- `knat` : 고마움 / (knat = ㅋ하나ㅌ)
- `snat` : 미안함 / (snat = 쉬나ㅌ)


## 대명사
- `kn` : 그 / (khn = ㅋ하ㄴ)
- `rt` : 그것 / (rt = 르ㅌ)
- `km` : 그녀 / (khm = ㅋ하ㅁ)
- `ti` : 그들 / (ti = ㅌ잇 = 팃)
- `e`  : 나 / (e = 에)
- `ak` : 너(당신) / (ak = 아ㅋ하)
- `te` : 우리 / (te = ㅌ에 = 테)
